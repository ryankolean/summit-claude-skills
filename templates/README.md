# Templates

Reusable starter files for Claude Code repos and personal configuration. Copy, rename, fill in the bracketed placeholders.

| Template | Purpose | Where it goes |
|---|---|---|
| [`personal-CLAUDE.md`](./personal-CLAUDE.md) | Global Claude Code operating standards for one user across every session | `~/.claude/CLAUDE.md` |

## Companion

- Repo-level `AGENTS.md` files live at the root of each repo, not in this templates directory. See [`AGENTS.md`](../AGENTS.md) for this repo's agent guide, and [agents.md](https://agents.md) for the open spec.
